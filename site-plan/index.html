Individual Project Site Plan

WDD 231 – Web Frontend Development I

Site Name

Linux Distro Catalog

This site name was selected because it clearly represents a catalog-style website that helps users compare Linux distributions for daily use.

Site Purpose

The purpose of this website is to help users understand the differences between popular Linux distributions and choose the one that best fits their needs. The site will provide information such as the distribution name, base system, desktop environment, target user level, and recommended use cases.

Scenarios

  • Which Linux distribution is best for beginners switching from Windows?
  • What Linux distributions are suitable for daily use on low-end hardware?

Color Scheme

The following colors will be used consistently throughout the site:

#0D47A1 – Used for headings, navigation, and accents
#F5F7FA – Used for page backgrounds and content areas

Typography

  • Montserrat – Used for headings and navigation
  • Roboto – Used for body text and descriptions
